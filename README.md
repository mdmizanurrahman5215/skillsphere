## 📖 Overview  
**SkillSphere** is a modern, responsive online learning platform that allows users to explore courses, view detailed content, and securely access protected learning materials. It focuses on clean UI, smooth navigation, and real-world authentication.

---

## 🚀 Core Features  

- 🔐 Authentication (Email/Password + Google Login)  
- 🔒 Protected Routes (Course Details access control)  
- 📚 Course Browsing & Details  
- 🔍 Search courses by title  
- 👤 User Profile with update functionality  
- 🎨 Fully Responsive Design  
- ⚡ Loaders, animations, and toast notifications  
- ❌ Custom 404 page & reload-safe routing  

---

## 🧱 Application Structure  

- **Navbar:** Dynamic auth state (Login/Register vs Avatar/Logout)  
- **Footer:** Contact info, social links, policies  

### Pages:
- Home  
- All Courses  
- Course Details (Protected)  
- Login / Register  
- My Profile
